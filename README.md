This is a Chrome extension that will block accounts for you on Twitter/X. 

To install, go to Manage Extensions and select Load Unpacked at the top. Select the downloaded folder and make sure to enable Pin to Toolbar.

The latest version is called Account Tracking. The new features are in the Options page that can be reached by right-clicking the Toolbar Pin.

Pop-Up Menu (toolbar):
Block Followers Button
Stop Button
Checkbox to Enable Tab Closing
Max Blocks textbox for custom number input
Total Blocked cache

How to Use Toolbar:
The Block Followers button can be used on URLs that end with /followers for example x.com/elonmusk/followers
You can input the number of blocks you want to occur in one run.
WARNING: Twitter/X has built-in rate limiting that will pause browser activities for a period of time if you perform more than about 50 blocks in 10 minutes
The Followers page will have about 70 followers on it. 

When you click Block Followers, the following will happen:
A new tab will be opened on the follower's account
The block function will be found and used
The tab will close by itself if the Enable Tab Closing checkbox is selected
The next follower on the list will be found and the tab will be opened.
Once the number of Max Blocks has been reached, an alert will pop-up signalling the number of blocks has been reached.

How to use Block Reply Accounts:
This button will be shown if the checkbox for it is selected in the Options menu, labeled "Block and add accounts that reply to posts".
The Block Reply Accounts button will be shown automatically on URLs that end with /status/1234567 at the top of the page. This applies to Quote posts and normal posts.
You may need to refresh the page to see the button.
The Block Reply Accounts button will open the three dot button and block the account from there.


How to use the Options Menu
At the top is a checkbox labeled "Automatically add blocked accounts as source accounts".
When this is enabled, any accounts that the extension blocks will be added to the list of Source Accounts labeled "Blocked Followers Per Account" on the left side.

Left Side:
By default it will show  the number of accounts that have been blocked using that account as a Source Account out of a blank textbox that can be filled in. 
The "Go to" button will open the account's Followers page on a new tab.
The "Complete" button will send the account to the "Completed Accounts" section on the Right Side. 
The other buttons can be used to categorize accounts by how many followers they have, although this is a manual process.

Right Side:
On the right side, there is a list of default Categories which are Less than 50, 51-500, 501-1000, 1001-5000, 5000+ and the Completed Accounts section below. 
Completed Accounts only have the "Undo" button which will send an account back to the Left Side.
Accounts in Categories have the "Go to" button for quick access to their Followers page andthe "Complete" button to send the account to the Completed category.

I recommend blocking 10-20 accounts per account and moving on for most accounts that you don't want to keep track of.
